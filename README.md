# E-Commerce MERN Application

A full-stack E-Commerce website built with the MERN Stack.

## Features

- User Authentication
- Admin Dashboard
- Product Management
- Shopping Cart
- Checkout
- Order Management

## Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB

## Installation

```bash
npm install
npm run dev
```
